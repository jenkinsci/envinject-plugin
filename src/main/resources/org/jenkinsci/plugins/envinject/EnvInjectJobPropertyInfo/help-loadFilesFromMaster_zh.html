如果啟用，從 controller 節點加載文件(屬性或腳本)。 如果不選中此選項，Jenkins 從 target 節點 (從/代理或主節點) 加載文件。 從 controller 加載文件可以讓您避免在擁有通用路徑之間發生衝突，例如當您的作業未附加到特定節點時 Windows 和 Linux 之間的問題。 無論如何，文件將從 target 節點處理。

從 EnvInject 2.0 開始，全局配置可以禁用此選項。 當選項已被全局配置禁用時，您將看到一條警告，以標記該選項已由作業配置但現在已禁用。 關閉該選項後 - 確認作業已重新配置 - 該選項將從 UI 中消失。

不建議在新作業中使用此選項，即使它已啟用。
