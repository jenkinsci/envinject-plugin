如果启用，从 controller 节点加载文件(属性或脚本)。 如果不选中此选项，Jenkins 从 target 节点 (从/代理或主节点) 加载文件。 从 controller 加载文件可以让您避免在拥有通用路径之间发生冲突，例如当您的作业未附加到特定节点时 Windows 和 Linux 之间的问题。 无论如何，文件将从 target 节点处理。

从 EnvInject 2.0 开始，全局配置可以禁用此选项。 当选项已被全局配置禁用时，您将看到一条警告，以标记该选项已由作业配置但现在已禁用。 关闭该选项后 - 确认作业已重新配置 - 该选项将从 UI 中消失。

不建议在新作业中使用此选项，即使它已启用。
