Evaluate the Groovy scripts that the the given environment variables contain and inject the key value pairs returned in their result maps into the environment variable pool.
Enter a comma/space separated list of environment variable names. Example: variable_name_of_script_foo, variable_name_of_script_bar
The Groovy scripts must return a Java map object.
You can access parameters and other environment variables through variables in the Groovy script.
This is especially useful to trigger Groovy scripts set by the SharedObjects plugin. Note: Populate Shared Objects must be enabled to access variables contributed by that plugin.
